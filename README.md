# Llanquihue Tour App

## Descripción
Aplicación Java para la agencia de turismo **Llanquihue Tour** que lee un archivo de datos, crea objetos Tour y los almacena en un ArrayList, mostrando la lista completa y aplicando filtros.


## Instrucciones para ejecutar
1. Abrir el proyecto en IntelliJ IDEA
2. Ejecutar la clase `ui.Main`
3. Ver los resultados en consola

## Funcionalidades
- Lectura de archivo `tours.txt` desde resources
- Almacenamiento en ArrayList
- Recorrido completo de la colección
- Filtrado por precio mínimo (> $70.000)
- Filtrado por tipo de tour (gastronómico)

## Formato del archivo tours.txt

nombre;ubicacion;precio;tipo
